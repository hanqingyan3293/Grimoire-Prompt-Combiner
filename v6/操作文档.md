# 魔导书 Grimoire v6.0 — 操作文档

## 一、安装与启动

### 绿色便携版
直接运行 Grimoire 6.0.0.exe，解压后自动启动。适合 U 盘携带、无需管理员权限。

### 安装版
运行 Grimoire Setup 6.0.0.exe，按向导完成安装。支持自定义安装路径，会自动创建桌面快捷方式。

### 开发模式（仅开发者）
`cmd
cd v6
npm install
npm run dev
`

## 三、标签库管理

### 3.1 浏览标签
左侧边栏显示三级标签分类：
- **类别** → 如 "角色"、"场景"、"风格"
- **子类** → 如 角色→"发型"、"表情"、"服装"
- **标签** → 如 发型→"long hair"、"ponytail"

### 3.2 搜索标签
在侧边栏搜索框中输入关键词（支持英文/中文），实时筛选匹配的标签。

也可按 **Ctrl+K** 快速聚焦搜索框。

### 3.3 添加标签到提示词
- **左键点击** — 添加到正向提示词，权重默认 1.0
- **Shift+左键点击** — 添加到负向提示词

### 3.4 标签操作
每个标签卡片上可进行：
- **♥ 收藏** — 将标签加入收藏夹
- **✏ 编辑** — 修改标签的中英文名称
- **🗑 删除** — 从库中移除标签

### 3.5 新增标签
点击侧边栏顶部的 **+Tag** 按钮，填写：
- 所属类别 / 子类（可新建）
- 英文名（必填）
- 中文名（可选）

### 3.6 导入导出
- 菜单栏：文件 → **导入标签库** / **导出标签库**
- 支持 JSON 格式，可在不同电脑间迁移标签库

## 四、提示词工作区

### 4.1 正向提示词区域
已添加的标签以卡片形式展示。每个卡片上可操作：
- **权重滑块** — 拖拽调节标签权重（0.1 ~ 2.0）
- **权重数值** — 直接点击数字输入精确值
- **× 移除** — 从提示词中移除该标签

### 4.2 负向提示词区域
操作方式同正向，存放不希望在画面中出现的内容标签。

### 4.3 工具按钮
- **📋 复制** — 将格式化提示词（含权重）复制到剪贴板
- **🔄 随机顺序** — 打乱标签排列
- **🔍 去重** — 移除重复标签
- **🗑 清空** — 清空当前提示词面板
- **✅ 质量词** — 自动在开头添加 "masterpiece, best_quality"

### 4.4 撤销与重做
- **Ctrl+Z** — 撤销上一步操作
- **Ctrl+Y** — 重做
- 最多支持 50 步撤销

## 五、AI 功能

### 5.1 配置 AI
1. 点击右侧面板的 **⚙ Settings** 标签
2. 找到 AI 设置区域
3. 填入配置：
   - **Provider** — 选择 "OpenAI Compatible"
   - **Endpoint** — 填入 API 基础地址，如 https://api.nonelinear.com/v1
   - **API Key** — 填入你的 API 密钥
   - **Model** — 填入模型名称，如 grok-3、gpt-4o
4. 点击 **保存**

### 5.2 AI 聊天
1. 切换到 **🤖 AI** 面板
2. 默认显示 **💬 Chat** 子标签
3. 在顶部下拉框选择模型
4. 点击 **⚙ 按钮** 可编辑 System Prompt（影响 AI 回答风格）
5. 在输入框中输入问题，按 **Enter** 发送
6. 点击 **🗑 按钮** 清空对话

### 5.3 AI 识图
1. 在 AI 面板切换到 **🖼 Vision** 子标签
2. **拖拽图片** 到虚线区域，或点击选择图片文件
3. 点击 **Analyze Image** 按钮
4. AI 分析后返回标签列表，每个标签显示：
   - 标签名
   - 分类
   - 置信度（百分比 + 颜色指示灯）
5. 对单个标签可操作：
   - **+** — 添加到正向提示词
   - **📑** — 保存到标签库
6. 对全部标签可使用：
   - **+ Prompt** — 一键全部添加到提示词
   - **+ Library** — 一键全部保存到标签库

> **注意：** 识图功能需要 API 支持 vision 模型。部分中转商可能不支持图片输入。

## 六、预设与历史

### 6.1 预设（Presets）
- 切换到 **💾 Presets** 面板
- 点击 **保存当前** 将当前提示词组合存为预设
- 点击预设名称即可一键加载

### 6.2 历史（History）
- 自动记录每次提示词组合变化
- 切换到 **🕐 History** 面板查看
- 点击任意历史记录可恢复

|--------|------|
| 语言 | 中文 / English |

| 自定义颜色 | 色轮选择主题强调色 |
| AI 配置 | Provider、Endpoint、API Key、Model |
