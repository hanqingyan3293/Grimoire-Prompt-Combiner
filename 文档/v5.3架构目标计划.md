## 一、版本概述

v5.3 是从 v5.2 源码全面重构的 Photoshop 风格版本。所有旧版本（v5 / v5.1 / v5.2）保持不变。

### 核心变更
- **PS 专业暗色界面**：4 级亮度主题（极暗/暗色/中暗/亮色）
- **标签化面板系统**：右侧提示词/历史/预设/图库/设置以标签页组织
- **面板折叠区域**：PS 风琴式 PanelSection 折叠/展开
- **底部状态栏**：面包屑路径 + 标签统计 + 提示词计数
- **全宽选项栏**：工作区/语言/字体/主题集成
- **工作区预设**：默认/紧凑/宽画布/双行 4 种布局

### 技术栈
- Electron + React 18 + TypeScript
- Zustand 状态管理（stores 全部保留未变）
- SQLite 数据持久化
- Vite 构建
- PS 风格纯 CSS 设计系统（无 UI 库依赖）

## 四、CSS 分层设计系统

### tokens.css — 设计令牌
```css
--ps-bg-deep:     #121212;  --ps-bg-canvas:   #1a1a1a;
--ps-bg-panel:    #1e1e1e;  --ps-bg-header:   #2d2d2d;
--ps-border:      #3c3c3c;  --ps-text-primary:#cccccc;
--ps-accent:      #4b9cd3;  --ps-accent-hover:#5dade2;
--sidebar-width:  250px;    --right-panel-width:320px;
```

### themes.css — 4 级亮度主题
- `darkest` (默认) — 最深黑底
- `dark` — 略亮
- `medium-dark` — 中灰
- `light` — 亮色

### components.css — PS 组件样式
- `.ps-panel` / `.ps-panel-header` — 面板容器+标题
- `.ps-tab-bar` / `.ps-tab` — 标签页系统
- `.ps-section` — 折叠区域
- `.ps-btn` — PS 风格按钮
- `.ps-input` — PS 风格输入框
- `.ps-icon-btn` — 图标按钮
- `.ps-tag` — 标签卡片
